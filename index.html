Настройка Windows

  • Встроенные приложения Windows

      PowerShell(От имени администратора)

      Удаление:


      Get-AppxPackage people | Remove-AppxPackage


      Get-AppxPackage | Remove-AppxPackage

      Восстановление:


      Get-AppxPackage -allusers | foreach {Add-AppxPackage -register "$($_.InstallLocation)\appxmanifest.xml" -DisableDevelopmentMode}

  • Оптимизация в Windows с помощью программы ПК Без Проблем.

      Отключить Антивирус Windows также можно с помощью него.

  • Режим Максимальной Производительности

      PowerShell(От имени администратора)


      powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

  • Повышенная точность указателя Мыши

      "Win + r" → control → Мышь.




      "Параметры указателя" → убрать галочку с "Включить повышенную точность".

  • Повышаем громкость Микрофона

      ПКМ по значку громкости → "Звуки". "Запись", выбор микрофона → ПКМ → Свойства. "Уровни" и выкручиваем на 100%.
